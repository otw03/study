오태원 Github 실습1

  1. Github에 프로젝트 생성 —> 프로젝트이름은 각자 정함

  1. 사용자 홈 폴더 안에 school 이라는 이름의 폴더를 만들고 그 안에서 clone받기

  1. clone받은 프로젝트 안에서 임의의 파일을 3개 이상 생성하고 push

  1. 사용자 홈 폴더 안에 home이라는 이름의 폴더를 만들고 새롭게 clone받기

  1. home안에 받은 내용에 하나의 파일은 수정, 하나의 파일은 삭제, 새로운 파일 하나 추가 -> push

  1. school 안에서 pull 받은 후 변경 내용이 반영 되었는지 확인

  1. school안에 받은 내용에 하나의 파일은 수정, 하나의 파일은 삭제, 새로운 파일 하나 추가 -> push

  1. home 안에서 pull 받은 후 변경 내용이 반영 되었는지 확인
